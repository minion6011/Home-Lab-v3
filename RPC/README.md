# ❓ What is this
This is the discord RPC integration for the music section of the Homelab. <br>
To use this integration you need to run RPC.py on your **client** while using the Homelab.
> [!note]
> Before running RPC.py remember to add the `CLIENT_ID` inside the file you can get it after creating an app on [Discord Developer Portal](https://discord.com/developers/applications)
